Алгоритм двумерной упаковки в полуограниченную полосу

Загрузка данных из файла (.xlsx | .txt)

Для работы необходимо создать в книге лист с названием "data" без кавычек. В ячейке A1 - пишем figure, В ячейке B1 - пишем count. Начиная с строки 2, вводим сплошной таблицей (без пустых строк) в столбец figure - размер фигуры, в столбец count - количество фигур. После окончания работы с файлом сохраняем книгу. Загружаем на сервер полученный файл при помощи "Загрузка данных из файла Excel". В случае удачной загрузки справа появится список размеров фигур и количество.
При нажатии на кнопку Start optimization! автоматически программа начнет НЕМЕДЛЕННЫЙ РАСЧЕТ и ВОЗВРАТ результата в файл эксель!
От размера подаваемых на вход данных будет зависеть время работы алгоритма, процесс оптимизации можно контролировать при помощи индикатора.
